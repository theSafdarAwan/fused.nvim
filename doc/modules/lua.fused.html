setup (user_configuration)
Loads the theme.

Parameters:

  • user_configuration table|nil configuration for theme.
load_plugin (name)
This lets you Load single plugin specified as parameter after the theme is loaded. Can be used in plugin configuration to load plugin Highlight groups only when plugin is loaded. Can act as a lazy loader.

Parameters:

  • name string name of the plugin.

Tables

Config
configuration for theme.

Fields:

  • flavour string name of the theme.
  • italics boolean to enable or disable italic font.
  • bg_transparent boolean to enable or disable transparent background.
  • custom table of custom higlight groups.
  • plugins_integration table of plugin names (name = boolean)
